vueCli #

作用命令
安装淘宝镜像 npm install -g @vue/cli 或者 yarn global add @vue/cli
查看版本vue --version
升级脚手架npm update -g @vue/cli 或者 yarn global upgrade --latest @vue/cli
升级某些插件vue upgrade [options] [plugin-name]

升级某个插件时的选项：

-t, --to version 升级 plugin-name 到指定的版本

-f, --from version 跳过本地版本检测，默认插件是从此处指定的版本升级上来

-r, --registry url 使用指定的 registry 地址安装依赖

--all 升级所有的插件

--next 检查插件新版本时，包括 alpha/beta/rc 版本在内

-h, --help 输出帮助内容

安装docker #

作用命令
官方安装脚本安装``` curl -fsSL https://get.docker.com
国内 daocloud 安装``` curl -sSL https://get.daocloud.io/docker

Compose 安装 sudo curl -L "https://github.com/docker/compose/releases/download/1.24.1/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
Compose 赋与可执行权限sudo chmod +x /usr/local/bin/docker-compose
Compose查看版本docker-compose --version
