# nvm

作用 命令
win安装 下载链接
linux安装 curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.37.2/install.sh | bash
查看所有node版本 nvm list
切换为某个node版本 nvm use 需要的版本
设置npm_mirror nvm npm_mirror https://npm.taobao.org/mirrors/npm/
设置node_mirror nvm node_mirror https://npm.taobao.org/mirrors/node/

# npm

作用 命令
安装淘宝镜像 npm install -g cnpm -registry=https://registry.npm.taobao.org
查看版本 npm -v

# yarn

作用 命令
安装 npm install -g yarn
安装淘宝镜像 yarn config set registry https://registry.npm.taobao.org -g
查看版本 yarn --version

# vueCli

作用 命令
安装淘宝镜像 npm install -g @vue/cli 或者 yarn global add @vue/cli
查看版本 vue --version
升级脚手架 npm update -g @vue/cli 或者 yarn global upgrade --latest @vue/cli
升级某些插件 vue upgrade [options] [plugin-name]

升级某个插件时的选项：

-t, --to version 升级 plugin-name 到指定的版本

-f, --from version 跳过本地版本检测，默认插件是从此处指定的版本升级上来

-r, --registry url 使用指定的 registry 地址安装依赖

--all 升级所有的插件

--next 检查插件新版本时，包括 alpha/beta/rc 版本在内

-h, --help 输出帮助内容

# 安装docker

作用 命令
官方安装脚本安装 curl -fsSL https://get.docker.com | bash -s docker --mirror Aliyun
国内 daocloud 安装 curl -sSL https://get.daocloud.io/docker | sh
查看版本 docker --version
Compose 安装 sudo curl -L "https://github.com/docker/compose/releases/download/1.24.1/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
Compose 赋与可执行权限 sudo chmod +x /usr/local/bin/docker-compose
Compose查看版本 docker-compose --version
